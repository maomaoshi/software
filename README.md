### 0x01 判断是否已登录
#### method post
#### url api/is_login

#### get
* filed: id
	+ 工号/学号
* filed: password
	+ 密码
* filed: captcha
	+ 验证码

#### return
```
{
	"status":"fail",
	"message":"no login"
}
{
	"status":"fail",
	"message":"captcha error"
}
{
	"status":"success",
	"message":"have logged in"
}

```
